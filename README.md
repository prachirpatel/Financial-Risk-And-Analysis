💰 Financial Risk and Analysis Web App
📌 Project Overview

This is a simple Financial Risk and Analysis Web Application built using Flask (Python) and PostgreSQL.
It helps users manage their income and expenses, track spending, and view financial data easily.

🚀 Features
🔐 User Registration & Login
🔑 Secure Password Hashing
➕ Add Income & Expenses
📊 View All Transactions
✏️ Update Records
❌ Delete Records
📅 Track by Date & Category
🛠️ Tech Stack

Backend: Python, Flask
Database: PostgreSQL
Frontend: HTML, CSS, Bootstrap
Other: JavaScript
